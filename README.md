# Tic Tac Toe Docker App

## Project Name
Tic Tac Toe Flask Containerized App

## Language Used
Python (Flask)

## Branch Strategy
- feature/* → development work
- dev → stable branch
- master → production

## CI/CD Explanation
GitHub Actions pipeline builds Docker image, deploys container, runs automated tests.

## How to Run Locally
pip install -r app/requirements.txt
python app/app.py

## How to Run via Docker Compose
docker compose up -d

## Testing Instructions
pytest tests/
